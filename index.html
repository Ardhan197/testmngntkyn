Cakupan Full Sync Google Sheet

Auto-sync mencakup: Karyawan, Absensi, Cuti, Shift, Assignment Shift, Resign, Blacklist, Master Cabang, Master Status, Pengaturan, Integrasi, Metadata Aplikasi, dan Index Storage.

Auto-sync aktif: setelah data disimpan, aplikasi membuat antrean dan mengirim full sync otomatis tanpa menunggu tombol Sync Manual.

Catatan: file gambar foto/KTP tetap disimpan di aplikasi dan backup JSON; ke Google Sheet dikirim status dan ukuran gambar agar cell Google Sheet tidak rusak.

Ambil dari URL Google Sheet: docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit. Link Apps Script bukan Spreadsheet ID.

Error SHEET_ID is not defined berarti kode Apps Script yang sedang ter-deploy belum punya variabel Spreadsheet ID. Salin ulang template Apps Script terbaru di bawah, isi ID Spreadsheet, lalu Deploy ulang Web App.
