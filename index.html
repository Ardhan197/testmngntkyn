Cakupan Database Google Sheet

Database Google Sheet mencakup: Karyawan, Absensi, Cuti, Shift, Assignment Shift, Resign, Blacklist, Master Cabang, Master Status, Pengaturan, Hak Akses/Login, Metadata Aplikasi, Maintenance, Gambar, dan Delete Log.

Foto karyawan dan KTP dikirim ke Google Drive melalui Apps Script; link file disimpan di Google Sheet agar bisa dibuka dari perangkat lain.

Ambil dari URL Google Sheet: docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit. Link Apps Script bukan Spreadsheet ID.

Foto dan KTP akan diupload ke Google Drive lalu linknya disimpan di Google Sheet.

Mode Google Sheet membutuhkan Spreadsheet ID, Web App URL Apps Script, deployment Web App dengan akses yang sesuai, dan izin Drive untuk upload foto/KTP. Local Storage hanya dipakai sebagai cache/offline fallback.

Restart & Kosongkan Data

Gunakan hanya saat maintenance besar. Fitur ini dipisah agar user bisa memilih reset lokal saja atau mengosongkan seluruh data aplikasi.

Zona Berisiko

Restart Ulang Lokal menghapus storage browser dan reload aplikasi. Karena server berada di localStorage browser ini, lakukan Backup JSON sebelum restart bila data ingin disimpan eksternal.

Kosongkan Seluruh Data mengosongkan data karyawan, absensi, cuti, shift, resign, blacklist, master, maintenance, dan pengaturan. Karena server adalah localStorage, tindakan ini menghapus data aplikasi di browser ini.
