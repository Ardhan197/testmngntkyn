Pengaturan Integrasi Google

Salin kode Apps Script smart di bawah ini ke file Code.gs. Script ini otomatis membuat Spreadsheet, tab, header, folder Drive, dan tetap membaca data walau urutan header berubah.

Status Koneksi

Local

Jumlah Baris Kode

582 baris

Target Database

Google Sheet + Drive

Langkah Setup

  1. Buka script.google.com, buat project baru.
  2. Paste kode Code.gs dari halaman ini.
  4. SPREADSHEET_ID dan DRIVE_FOLDER_ID boleh dikosongkan/placeholder karena script bisa membuat otomatis.
  5. Deploy sebagai Web App. Pada akses, pilih sesuai kebutuhan operasional.
  6. URL Web App sudah dipasang pada frontend. Setelah mengganti Code.gs, lakukan Update Deployment / New version.
  7. Jalankan aplikasi. Saat koneksi pertama, tab Sheet, header, folder Drive, dan subfolder upload akan dibuat otomatis.
