Restart & Kosongkan Data

Gunakan hanya saat maintenance besar. Restart lokal hanya membersihkan cache browser, sedangkan Kosongkan Seluruh Data melakukan hard reset data bisnis, histori, master, pengaturan, dan cloud jika online.

Zona Berisiko

Restart Ulang Lokal menghapus storage browser dan reload aplikasi. Data aplikasi dapat memakai cache lokal/offline, tetapi sesi login tidak lagi disimpan permanen. Lakukan Backup JSON sebelum restart bila data ingin disimpan eksternal.

Kosongkan Seluruh Data melakukan hard reset: data karyawan, shift, absensi, histori, cuti, payroll, resign, blacklist, master, maintenance, hak akses, pengaturan, sync log, dan cache akan dikosongkan. Jika Web App aktif, isi Google Sheet juga dihapus sampai body tabel kosong. Setelah reset, aplikasi membuat ulang akun owner default agar sistem tidak terkunci.
